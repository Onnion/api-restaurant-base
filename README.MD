# install pre-requisites

### Typescript
```
sudo apt-get node-typescript && npm i typescript -g
```

### MongoDB
```
sudo apt install -y mongodb
```


### PM2
```
npm i pm2 -g
```

# Install dependencies
```
npm i
```

## Start app

### start app
```
npm run dev
```

### stop app
```
pm2 stop ...
```
