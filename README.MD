# install pre-requisites

### Typescript
```
sudo apt-get node-typescript && npm i typescript -g
```

### MongoDB
https://www.digitalocean.com/community/tutorials/como-instalar-o-mongodb-no-ubuntu-16-04-pt


### PM2
```
npm i pm2 -g
```

# Install dependencies
```
npm i
```

## Start app

### start app
```
sudo npm run dev
```

### stop app
```
pm2 stop ...
```
